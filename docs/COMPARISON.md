## 架构对比

### NotchNook
```
单体架构，功能聚合
├── 核心状态管理
├── 功能模块（媒体、日历、笔记...）
├── UI 组件（波浪、渐变...）
└── 系统集成（蓝牙、摄像头...）
```

### X Island
```
模块化架构，职责清晰
├── SessionManager（核心状态机）
├── SocketServer（通信层）
├── ZeroConfigManager（配置层）
└── Views（UI 层）
```

### NookX
```
多服务架构，功能丰富
├── Services（音乐、天气、日历...）
├── Managers（缓存、监控...）
├── Views（WKWebView + SwiftUI）
└── Coordinators（协调器）
```

### X Nook（规划）
```
混合架构，兼顾灵活和简洁
├── Core（窗口、检测、切换）
├── Features（Widget 模块）
├── Managers（服务管理器）
└── UI（动画、组件）
```

## X Nook 的差异化优势

1. **开源免费** — MIT 许可，社区共建
2. **应用切换** — 与 X Island 联动，双指滑动切换
3. **模块化设计** — 功能可选，按需启用
4. **现代架构** — Swift Concurrency、Observation
5. **学习友好** — 代码清晰，文档完善
