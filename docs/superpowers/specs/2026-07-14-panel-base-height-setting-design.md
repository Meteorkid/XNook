# 展开面板基础高度设置

## 目标

恢复面板高度调节入口，同时保留内容驱动的自适应高度，避免最近任务或小组件被裁切。

## 交互

- 在「显示 → 面板尺寸」中增加「面板基础高度」滑块。
- 范围为 300–900pt，默认 400pt，步长 10pt。
- 文案说明：该值控制展开面板的最小高度；内容超出时会自动继续增高。

## 行为

- 新增 `panelBaseHeight` 的 UserDefaults 默认值与 `@AppStorage` 绑定。
- 展开面板高度为 `max(内容计算高度, panelBaseHeight)`。
- `NotchWindow` 继续负责屏幕边界保护；不会再使用固定高度上限裁切内容。

## 验收与测试

- 内容较少时，面板高度至少等于用户设置的基础高度。
- 增加最近任务记录时，超过基础高度后每条记录仍增加一行高度。
- 设置入口、默认值和本地化文案均可用。
